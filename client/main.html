Introduction

For centuries we’ve done most of our learning by reading static text on paper. Despite advances in computing, we’re still used to reading static text on our computers (pdf, doc, etc.). We can do better. What if we could interact with the notes we read? What if given a circuit diagram, we could explore what would happen if we changed all the various parameters?
